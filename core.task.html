Parameters

  • message – message to be sent.

  • priotity – the priority of the message. If there is another message on the output stream and the priority is lower than it, the message will be blocked.

  • message – optional message to be given with the reward

core.task.on_state_changed(condition)[source]¶

Decorator to capture a StateChanged event. Its condition is a function that takes the tasks state (or the world state and the task state, in that order, if the task has a world parameter) and checks for any condition on those state variables. Notice that the argument is the state instance variable within the task and not the task itself.
